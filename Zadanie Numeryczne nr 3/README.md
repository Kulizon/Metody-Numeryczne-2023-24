Do uruchomienia programu niezbędne są dwie paczki:
- numpy
- matplotlib

Można je zainstalować przy użyciu polecenia:
```
pip install numpy
pip install matplotlib
```

Następnie wystarczy uruchomić program w dowolnym interpreterze.
